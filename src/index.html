Distance Diffuse
Distance Diffuse blends each object with a mirrored version of itself (opposite horizontal direction) depending on distance. Close objects become wider/more diffuse, while far objects stay mostly direct.

Threshold: distance where the blend reaches 100% direct.
Curve: transition shape (faster or slower near-field blend).
Spread from Distance
Spread from Distance replaces object_size spread with a distance-driven spread value.

Formula: spread = (1 - distance/range)^curve, clamped to [0, 1], then mapped to the global spread min/max range.

Range: distance where spread reaches 0.
Curve: transition shape (higher = less spread near listener, more collapse farther away).
Trails
Trails show each object's recent movement history in the 3D view.

Points: maximum number of stored positions per object.
Duration: how long trail points stay visible before fading out.
